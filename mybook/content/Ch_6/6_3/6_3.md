# 6.3 FIRST PASSAGE PROBABILITIES


In a typical Markov chain, we frequently observe that states are left and re-entered again and again. We have developed a means of computing the probability $\mathrm{p}^{(\mathrm{n})}{ }_{\mathrm{ij}}$ that a system will leave state i and be in state j after n transitions. But this does not give us any information about whether the system entered state j at any time before the n -th transition. Suppose we are specifically interested in the probability that a system leaves state $i$ and enters state j for the first time after n steps. This is called the first passage probability, and it is clearly related to the n-step probability. However, we must exclude all the ways in which the system may have entered state j before the n -th transition. For example, if we know the probability of a cloudy day on the slopes being followed three days later by a sunny day, we realize that this three-step transition can occur in several ways:

$$
\text{cloudy → cloudy → cloudy → sunny} \\
\text{cloudy → cloudy → sunny → sunny} \\
\text{cloudy → sunny → sunny → sunny} \\
\text{cloudy → cloudy → snowy → sunny} \\
\text{cloudy → snowy → snowy → sunny} \\
\text{cloudy → snowy → sunny → sunny} \\
\text{cloudy → sunny → snowy → sunny} \\
\text{cloudy → sunny → cloudy → sunny} \\
\text{cloudy → snowy → cloudy → sunny} \\
$$

The probability $\mathrm{p}^{(3)}{ }_{21}$ accounts for all of these possible paths. By contrast, the first passage probability will account for only those paths in which a sunny day does not occur until the third step. Thus, we want to measure the probability that one of the following paths will occur:

$$
\text{cloudy → cloudy → cloudy → sunny}\\
\text{cloudy → cloudy → snowy → sunny}\\
\text{cloudy → snowy → snowy → sunny}\\
\text{cloudy → snowy → cloudy → sunny}\\
$$

The first passage probability $\mathrm{f}^{(\mathrm{n})}{ }_{\mathrm{ij}}$ is computed as $\mathrm{p}^{(\mathrm{n})}{ }_{\mathrm{ij}}$ minus the probabilities of the n -step paths in which state j occurs before the n -th step. In one transition, $\mathrm{p}^{(1)}{ }_{\mathrm{ij}}=\mathrm{f}^{(1)}{ }_{\mathrm{ij}}$. Then for larger n ,

$$
\begin{aligned}
& f^{(2)}{ }_{i j}=p^{(2)}{ }_{i j}-f^{(1)}{ }_{i j} p^{(1)}{ }_{j j} \\
& f^{(3)}{ }_{i j}=p^{(3)}{ }_{i j}-f^{(1)}{ }_{i j} p^{(2)}{ }_{j j}-f^{(2)}{ }_{i j} p^{(1)}{ }_{j j} \\
& \cdot \\
& \cdot \\
& f^{(n)}{ }_{i j}=p^{(n)}{ }_{i j}-f^{(1)}{ }_{i j} p^{(n-1)}{ }_{j j}-f^{(2)}{ }_{i j} p^{(n-2)}{ }_{j j}-\ldots-f^{(n-1)}{ }_{i j} p^{(1)}{ }_{j j}
\end{aligned}
$$

or, more succinctly,

$$
f^{(n)}{ }_{i j}=p^{(n)}{ }_{i j}-\sum_{k=1}^{n-1} f^{(k)}{ }_{i j} p^{(n-k)}{ }_{j j}
$$

Returning to our example, we will illustrate $\mathrm{f}^{(\mathrm{n})}{ }_{\mathrm{ij}}$ for $\mathrm{n}=1,2$, and 3 .

And, just to check our intuition, let us re-examine $f^{(3)}{ }_{21}$. This should be $f^{(3)}{ }_{21}=p^{(3)}{ }_{21}$

$$
\begin{aligned}
& \text { - Probability (cloudy → cloudy } \text { → sunny } \text { → sunny) } \\
& \text { - Probability (cloudy → sunny } \text { sunny } \text { sunny) } \\
& \text { - Probability (cloudy → snowy } \text { → sunny } \text { → sunny) } \\
& \text { - Probability (cloudy → sunny } \text { snowy } \text { sunny) } \\
& \text { - Probability (cloudy } \text { sunny } \text { s cloudy } \text { sunny) } \\
& \text { = } 0.407-0.098-0.098-0.035-0.020-0.004 \\
