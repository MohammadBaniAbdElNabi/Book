# 4.7.2 A Simple Example


Consider the following example problem, which is illustrated in Figure 4.9:

$$
\begin{array}{ll}
\operatorname{maximize} & \mathrm{z}=\mathrm{x}_{1}+2 \mathrm{x}_{2} \\
\text { subject to } & 2 \mathrm{x}_{1}+\mathrm{x}_{2} \leq 2 \\
& \mathrm{x}_{1}, \mathrm{x}_{2}=0 \text { or } 1
\end{array}
$$

![](https://cdn.mathpix.com/cropped/2006c750-dc2a-4c1f-9672-416eca5f2ff4-211.jpg?height=651&width=612&top_left_y=283&top_left_x=566)

**FIGURE 4.9** <br>
Simple Lagrangian problem.

Observe that, if a $0-1$ problem does not have any constraints, it is trivial to maximize. That is, if the objective function coefficient $\mathrm{c}_{\mathrm{j}}$ is positive, then set $\mathrm{x}_{\mathrm{j}}=1$; otherwise, set $\mathrm{x}_{\mathrm{j}}=0$. We can express the problem in Lagrangian form as:

$$
\underset{\mathrm{u} \geq 0}{\operatorname{minimum}} \underset{\mathrm{x} \in(0,1)}{\operatorname{maximum}} \quad \mathrm{x}_{1}+2 \mathrm{x}_{2}-\mathrm{u}\left(2 \mathrm{x}_{1}+\mathrm{x}_{2}-2\right)
$$

We begin with $\mathrm{u}=0$, and note the maximum of the problem is $\mathrm{L}(\mathrm{x}, 0)=3$ with $\mathrm{x}_{1}, \mathrm{x}_{2}=1$. However, this point violates the constraint, so we substitute these values of x into the Lagrangian, and consider the result as a function only of $u$.

$$
\underset{u \geq 0}{\operatorname{minimum}} 3-u
$$

This function can be minimized by choosing $u$ as large as possible. We could try $u=5$, for example; and when we substitute this value into the original Lagrangian, we get:

$$
\begin{aligned}
& \underset{\quad \underset{x=(0,1)}{\operatorname{maximum}} x_{1}+2 x_{2}-5\left(2 x_{1}+x_{2}-2\right)}{\substack{x=(0,1) \\
=\underset{x}{\operatorname{maximum}}-9 x_{1}-3 x_{2}+10}}
\end{aligned}
$$

The optimal solution to this problem is to set both decision variables to zero. The corresponding function value is $\mathrm{L}(\mathrm{x}, 5)=10$. This time, when we substitute x into the Lagrangian in terms of $u$, we find:

$$
\underset{u \geq 0}{\operatorname{minimum}} 0-u(0-2)=2 u
$$

![](https://cdn.mathpix.com/cropped/2006c750-dc2a-4c1f-9672-416eca5f2ff4-212.jpg?height=675&width=682&top_left_y=281&top_left_x=533)

**FIGURE 4.10** <br>
$\mathrm{L}\left(\mathrm{x}^{*}, \mathrm{u}\right)$ as a function of u .

This subproblem tells us to decrease $u$ as far as possible. We already know however that when $\mathrm{u}=0$, it will tell us to increase u . So, the correct value of u must lie somewhere between these two extremes.

Observe that, for any value of u , we can solve for x and find the value of $\mathrm{L}(\mathrm{x}, \mathrm{u})$. Figure 4.10 illustrates what we have learned so far about $\mathrm{L}(\mathrm{x}, \mathrm{u})$ as a function of u .

Recall that we want to minimize $L(x, u)$ as a function of $u$. When $u=0$, we found $x=(1,1)$ and the function was defined by the decreasing line as $u$ increases. This expression is valid as long as $x=(1,1)$; but then at some point, the optimal solution for $x$ changes, and we get a new linear function describing $\mathrm{L}(\mathrm{x}, \mathrm{u})$. We now know what that linear function is when $\mathrm{u}=0$ and $\mathrm{u}=5$, yet we do not know how it behaves in between these two points. The two line segments in Figure 4.10 represent our best guess at the moment. In particular, it looks as if the minimum value of the Lagrangian will be found when $\mathrm{u}=1$, so we try that next.

Substituting $\mathrm{u}=1$ into the Lagrangian gives:

$$
\begin{aligned}
& \operatorname{maximum} \mathrm{x}_{1}+2 \mathrm{x}_{2}-1\left(2 \mathrm{x}_{1}+\mathrm{x}_{2}-2\right) \\
& \quad \mathrm{x}=(0,1) \\
& =\operatorname{maximum}-\mathrm{x}_{1}+\mathrm{x}_{2}+2 \\
& \quad \mathrm{x}=(0,1)
\end{aligned}
$$

The maximum of this function is $\mathrm{L}(\mathrm{x}, 1)=3$ when $\mathrm{x}=(0,1)$. If we substitute $\mathrm{x}=(0,1)$ into the original function, we get:

$$
\mathrm{L}(\mathrm{x}, \mathrm{u})=2-\mathrm{u}(-1)=2+\mathrm{u}
$$

This new section of the Lagrangian is added to the previous approximation to get the function illustrated in Figure 4.11.

From this function, we obtain a new estimate of the minimum value of $u=0.5$. Once again, we substitute this value into the Lagrangian and solve for x .

![](https://cdn.mathpix.com/cropped/2006c750-dc2a-4c1f-9672-416eca5f2ff4-213.jpg?height=518&width=566&top_left_y=285&top_left_x=593)

**FIGURE 4.11** <br>
$\mathrm{L}\left(\mathrm{x}^{*}, \mathrm{u}\right)$ as a function of u .

$$
\begin{aligned}
& \operatorname{maximum} x_{1}+2 x_{2}-0.5\left(2 x_{1}+x_{2}-2\right) \\
& \quad x=(0,1) \\
& =\underset{x=(0,1)}{\operatorname{maximum}} 1.5 x_{2}+1
\end{aligned}
$$

The maximum of $\mathrm{L}(\mathrm{x}, 0.5)=2.5$ occurs at $\mathrm{x}=(0,1)$ or $\mathrm{x}=(1,1)$. It is easy to verify that this is the true minimum of the Lagrangian. That is, we will not find any new solutions that we have not already described in Figure 4.11.
Let us summarize a number of very useful properties of the Lagrangian, and indicate how we can make use of these properties.

- The Lagrangian method always finds an integer solution, although the solution found is not necessarily feasible.
- If the solution, $x^{\mathrm{I}}$ is feasible, and if the original function, $z^{\mathrm{I}}$ at $x^{\mathrm{I}}$ is equal to the value of the Lagrangian, then $x^{I}$ is optimal for the original integer problem.
- Most important, if $\mathrm{z}^{*}$ is the solution of the relaxed $\mathrm{LP}, \mathrm{L}(\mathrm{x}, \mathrm{u})$ is the optimal solution to the Lagrangian, and $\mathrm{z}^{\mathrm{I}}$ is the (unknown) optimal integer function value, then

$$
\mathrm{z}^{\mathrm{I}} \leq \mathrm{L}(\mathrm{x}, \mathrm{u}) \leq \mathrm{z}^{*}
$$

A proof of these relationships can be found in Fisher (1985).
In other words, the value of the Lagrangian always gives a bound on the optimal integer solution that is at least as good as the LP. Therefore, if we use the Lagrangian instead of the LP in any branch-and-bound algorithm, we may get better results. The LP bound is never better than the bound from the Lagrangian. In our simple example problem, the optimal integer function value $\mathrm{z}^{\mathrm{I}}=2$ when $\mathrm{z}^{\mathrm{I}}=(0,1)$. The LP solution occurs at $\mathrm{x}=(0.5,1)$ with $\mathrm{z}^{*}=2.5$. The LP and the Lagrangian both give the same upper bound.
