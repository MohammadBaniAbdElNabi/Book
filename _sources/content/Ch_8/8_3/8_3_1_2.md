# 8.3.1.2 Average Waiting Time

For every customer $i$ in the system, compute

$$
\begin{aligned}
\mathrm{W}_{\mathrm{i}} & =\text { Waiting time }=\text { Time service begins }- \text { Arrival time } \\
& =\text { Time in system }- \text { Service time }
\end{aligned}
$$

Then accumulate the sum of these $\mathrm{W}_{\mathrm{i}}$ and divide by the number of customers N :

$$
\text { Average waiting time }=\frac{\left(\sum_{\mathrm{i}=1}^{\mathrm{N}} \mathrm{~W}_{\mathrm{i}}\right)}{\mathrm{N}}
$$

In the simulation, initialize the sum to zero; then at every event corresponding to service beginning (or a departure event), compute the $\mathrm{W}_{\mathrm{i}}$ for this customer and add it to the sum. At the end of the simulation, perform the division. In our example, from Table 8.2 again, we obtain the waiting time sum:

$$
0+0+2+3+3+0+0+2+0+4=14
$$

Then the average waiting time for these ten programs is 14/10= 1.4 time units.
